## 🔧 Easy Customization

### Change Colors
Edit `index.html` → find CSS colors:
```
#5f9eff  → Blue (Primary)
#ff6b7a  → Pink (Heart)
#05070a  → Dark background
```

### Add New System
1. Copy `pages/heart.html`
2. Rename to `system-name.html`
3. Change content + quiz
4. Add link in `index.html`

### Edit Quiz Questions
Open any `pages/*.html` → find `<form id="quizForm">`

## 📞 Troubleshooting

**Q: Pages not loading?**
A: Clear cache (Ctrl+Shift+Del) + check file paths

**Q: GitHub Pages taking time?**
A: Wait 5 minutes, then refresh

**Q: Quiz scores not saving?**
A: Check if localStorage is enabled

**Q: Want to test locally?**
A: `python -m http.server 8000` then `localhost:8000`
